## 🔍 发现的问题

### 严重问题 (Priority: High)
1. **前端未使用真实 API** - 所有组件仍使用 mock 数据
2. **登录状态管理缺失** - 无法在刷新后保持登录
3. **token 存储未实现** - localStorage 未使用
4. **订单状态流转 API 缺失** - 核心业务流程无法完成

### 中等问题 (Priority: Medium)
5. **WebSocket 未实现** - 无实时通知
6. **图片上传未实现** - 商品/评价无法上传图片
7. **支付接口缺失** - 订单无法真实支付
8. **地址管理 API 未实现** - 无法管理收货地址
9. **评价系统 API 未实现** - 无法评价订单
10. **优惠券系统不完整** - 无法领取使用优惠券

### 轻微问题 (Priority: Low)
11. **shared Header 未使用** - 四个 App 未迁移到共享组件
12. **设计 token 硬编码** - 未使用 CSS 变量
13. **错误处理不完善** - API 错误未友好展示
14. **加载状态缺失** - 无 loading 提示
15. **性能优化未做** - 无缓存、无懒加载
